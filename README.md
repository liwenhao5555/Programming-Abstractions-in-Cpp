Notes for macOS (Apple Silicon or Intel):

- Makefiles auto-detect architecture via `ARCH ?= $(shell uname -m)`.
- Graphics examples require Java. Recommended: `brew install openjdk@17`.
- Ensure Java is on PATH, e.g. add `export PATH="/opt/homebrew/opt/openjdk@17/bin:$PATH"` to `~/.zshrc`.
